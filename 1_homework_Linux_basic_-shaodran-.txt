1) Посмотреть где я
pwd

2) Создть папку
mkdir folder_1

3) Зайти в созданную папку
cd folder_1

4) Создать 3 папки
mkdir folder{11,12,13}

5) Зайти в любую папку
cd folder_12

6) Cоздать 5 файлов (3 txt, 2 json)
touch text{1,2,3).txt json{1,2}.json

7) Создать 3 папки
mkdir folder{21,22,23}

8) Вывести содержимое папки. (Без атрибута "а", нам нужно содержимое папок, уровни выше нас не интересуют)
ls -l

9) Открыть любой тхт файл
vim text1.txt

10) Написать что-то в файл
//после нажатия кнопки "i" входим в режим редактирования и пишем текст
//ESC выходим из режима редактирования 

11) Сохранить и выйти
:wq

12) Выйти из папки на уровень выше
cd ..

13) Переместить любые созданные ранее 2 файла в другую папку
mv folder_12/{text1.txt,text2.txt}  folder_11

14) Скопировать любые созданные ранее 2 файла в другую папку
cp folder_12/{json1.json,json2.json} folder_13


15) Найти файл по имени (Например text3.txt) Команда отобразит относительный путь файла
find -name "text3.txt"

16) Посматривать содержимое в реальном времени.(Мониторинг файла и отображение новых строк)
tail -f folder_11/text1.txt 
(с функцией поиска мониторит файл на содержание строк с текстом "hello" и отображает только их в реальном времени)
tail -f folder_11/text1.txt | grep hello

17) Вывести несколько первых строк из текстового файла (Например, 3 первые строки)
head -3 folder_11/text1.txt

18) Вывести несколько последних строк из текстового файла (Например, 3 последние строки)
tail -3 folder_11/text1.txt

19) Просмотреть содержимое длинного файла через less (Отображает часть, что помещается в окно, можно пролистывать)
less folder_11/text1.txt
